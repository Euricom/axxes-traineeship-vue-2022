## Slides

The slides are build with [slidev](https://sli.dev/) and are written in markdown.
 
### Slidev Installation

```bash
# install slidev
cd clides
npm install  # or yarn
```

### View as slide show

```bash
# Open slide deck
npx slidev fundamentals.md
```
